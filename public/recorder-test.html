5. Manual Picker Test Area

Instructions

Use the "Select Control" button in the recorder toolbar to manually pick elements below:

  • Hover over any element to see its confidence score
  • Click to capture the element identity
  • Press Escape to cancel picker mode

✅ Expected Test Results

High Confidence (85-99%):
  • data-adoption-id → 99%
  • data-testid, data-test, data-cy → 95%
  • Stable IDs → 90%
  • Name attributes → 85%
Medium Confidence (70-84%):
  • ARIA labels → 82%
  • Role + text → 78%
  • Label associations → 76%
  • Placeholder text → 70%
Low Confidence (<70%):
  • Generated IDs → 50-60% (penalty applied)
  • CSS selectors → 55%
  • XPath fallback → 40%
  • Duplicate text → Requires disambiguation
Expected Behaviors:
  • ✅ Low confidence captures trigger confirmation dialog
  • ✅ "Reselect Control" launches picker mode
  • ✅ Picker shows real-time confidence on hover
  • ✅ Picker click prevents actual page interaction
  • ✅ Escape key cancels picker mode
  • ✅ Password and card fields are masked
  • ✅ Multiple selector fallbacks stored per element
