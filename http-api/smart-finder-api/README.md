# Smart Finder API

Independent API for saving AI healing suggestions when a control is not found.

## Run

- Dev: `npm run smart-finder-api:dev`
- Start: `npm run smart-finder-api:start`

## Endpoint

- `POST /v1/healing-suggestions`
- `OPTIONS /v1/healing-suggestions`
- `GET /health`

This API mirrors the save behavior currently exposed by `app/api/guided-workflow-player/healing-suggestions/route.ts` (POST path).
